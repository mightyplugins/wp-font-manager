Add google font to your site easily.

== Description ==

Managing google font is never been easier. Previously you need to put google font by editing `functions.php`. But now you can add add fonts and subsets visually. Themes and plugins developer can easily integrate with there theme and plugin.

== Installation ==

1. Upload the entire `wp-font-manager` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'WP Font Manager' page at admin sidebar. Search font and add them to your site. After adding font you need to add font family to your css file or at 'Additional CSS' in customize panel.
