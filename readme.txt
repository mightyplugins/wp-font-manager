Build page visually with drag and drop builder.

== Description ==

Building, designing and managing a WordPress page is not easy. Editing manging shortcode at WordPress editor like a painful work. That's why we build Mighty Builder. This page builder will make your work enjoyable and more productive.

Mighty Builder is 100% pure drag and drop page builder. To use this page builder you don't need any knowledge about coding. Just install this plugin and start building WordPress site. This page builder is 100% responsive and mobile friendly. It's using bootstrap grid system.

= Elements =
* Text
* Heading
* Single Image
* Icon
* Space
* Line
* Alert
* Button
* Progress Bar
* Count Up
* Embed (Support: Youtube, Vimeo, etc [Full list](https://codex.wordpress.org/Embeds))
* Tabs
* Accordion

== Installation ==

1. Upload the entire `mighty-builder` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Active Page Builder' button in your WordPress page editor, just after page title.
